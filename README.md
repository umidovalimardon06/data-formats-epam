# JSON (JavaScript Object Notation)

## Overview

**JSON (JavaScript Object Notation)** is a lightweight data representation format used for storing and exchanging structured data. It is easy for humans to read and write, and easy for machines to parse and generate.

---

## Purpose

JSON is primarily used for **interchanging data between a client and a server**. It has become the standard format for modern web communication.

---

## Common Usage

* Data exchange between systems
* API communication
* Standard request and response format for **REST APIs**

---

## Data Types Supported by JSON

JSON supports the following data types:

* **String**
* **Number**
* **Boolean**
* **Null**
* **Object**
* **Array**

---

## Examples

### Numbers

```json
18
```

```json
"18"
```

> Numbers without quotes are treated as numeric values, while quoted numbers are treated as strings.

---

### Objects

```json
{
  "name": "Alice",
  "age": 25
}
```

---

### Arrays

```json
[
  "apple",
  "banana",
  "orange"
]
```

---

## Summary

JSON is a simple, efficient, and widely adopted format for data representation and communication, especially in web applications and RESTful APIs.
